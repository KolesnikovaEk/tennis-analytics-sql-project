# Tennis Analytics SQL Project

## О проекте
Проект представляет собой реляционную базу данных для хранения и анализа информации о профессиональном большом теннисе.  
В базе собраны данные об игроках, турнирах, матчах, статистике выступлений, тренировочных базах, спонсорах и команде сопровождения спортсменов.

Основная цель проекта — спроектировать удобную и расширяемую структуру данных, которая позволяет хранить связанную спортивную информацию и выполнять аналитические SQL-запросы.

## Что реализовано
- спроектирована структура реляционной базы данных;
- выделены основные сущности предметной области и связи между ними;
- разработана ER-диаграмма;
- созданы таблицы, первичные и внешние ключи;
- подготовлены SQL-скрипты для создания схемы и заполнения базы данными;
- реализованы SQL-запросы и представления для анализа игроков и матчевой статистики.

## Предметная область
База данных хранит информацию о:
- теннисистах и их характеристиках;
- тренировочных базах;
- команде сопровождения игрока;
- спонсорах;
- турнирах;
- матчах;
- статистике игроков по матчам;
- счёте по сетам.

## Структура проекта

- **advanced_part/** — представления, функции, индексы и расширенные SQL-решения  
- **base_part/** — создание таблиц, ограничения и базовая структура базы данных  
- **data/** — SQL-скрипты для заполнения базы тестовыми данными  
- **schemas/** — концептуальная, логическая и физическая схемы базы данных  
- **README.md** — описание проекта
